About Snippet Generator

I created a VS Code Extension for Polymer 1 and found the process of creating the actual snippets tedious. I developed a python script to generate a snippet from code I had pasted onto my clipboard. After this I thought it would be useful to share this for anyone else creating snippets and I decided to create a flask web app.

Please see the Creating your own snippets documentation to get started.

If you find any issues or have some cool additions, please contribute: Github Repo

*Please note I do not store your code, If security is a concern, please clone the repo and run locally.
